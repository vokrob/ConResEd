# ConResEd
*Конвертер и редактор резюме*

# Пользователь
Анна, 23 года, начинающий специалист (HR, маркетолог или IT-джуниор). Активно ищет работу, откликается на вакансии на hh.ru и «Хабр Карьере». У неё уже есть резюме, наспех сделанное в Word. Она боится, что из-за плохого оформления рекрутеры не воспримут её всерьёз. Ей нужно быстро «пересобрать» своё резюме, подобрать стильный шаблон, заполнить его и сохранить в PDF, не тратя время на изучение сложных редакторов.

Дмитрий, 29 лет, опытный специалист (middle+ разработчик или системный аналитик). У Димы есть резюме на hh.ru, которое он обновлял годами: там огромный список проектов, стек технологий и достижения. Но Дима потерял доступ к своему аккаунту, а перепечатывать 10 лет опыта вручную он не готов. Ему нужен инструмент, который заберет его "простыню текста" с hh.ru и разложит по полочкам в аккуратный документ.

# Функции приложения
1. Парсинг резюме: Загрузка данных по ссылке (например, с hh.ru) — приложение само заполнит поля (ФИО, опыт, навыки), чтобы пользователю не пришлось вводить всё вручную.

2. Редактор резюме: Удобное редактирование текста прямо в полях.

3. Библиотека шаблонов: Выбор визуального оформления из 3-5 готовых стилей с возможностью предпросмотра до применения.

4. Сохранение и экспорт:
- Автосохранение черновиков в личном кабинете.
- Возможность создавать несколько версий резюме (под разные вакансии).
- Экспорт в PDF одним кликом с идеальным сохранением верстки.

5. Хранение шаблонов на бэкенде.
