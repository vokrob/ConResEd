# ConResEd

Конвертер и редактор резюме

## Что сейчас есть
- Backend на Flask
- Регистрация пользователя
- Авторизация пользователя
- PostgreSQL с инициализацией тестовых данных
- Страницы с шаблонами резюме
- Сохранение информации в заполненных полях при переходах между шаблонами
- Frontend на React
- Интеграция UI -> Backend -> DB

## Запуск
```bash
docker compose up --build
```

После запуска: [http://localhost:8080](http://localhost:8080)

## Как работает
1. Перейдите на [http://localhost:8080](http://localhost:8080).
2. В блоке "Регистрация" заполните поля и нажмите "Зарегистрироваться".
3. При успешной регистрации API вернет `message` и данные пользователя.
4. В блоке "Вход" выполните авторизацию с зарегистрированными данными.
5. В блоке "Текущая сессия" проверьте, что отображается текущий пользователь, и нажмите "Выйти".
