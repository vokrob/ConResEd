# ConResEd

Конвертер и редактор резюме

## Что сейчас есть
- Backend на Flask
- Frontend на React
- Регистрация и авторизация пользователя
- PostgreSQL с инициализацией тестовых данных
- Страницы с шаблонами резюме
- Сохранение информации в заполненных полях при переходах между шаблонами
- Генерация QR-кодов для быстрого доступа к резюме с мобильных устройств
- Сохранение резюме в личный кабинет с сортировкой и поиском по имени
- Конвертация резюме в файлы формата PDF
- Парсинг DOCX и PDF файлов с текстом в поля шаблонов
- Оптимизация интерфейса приложения под мобильные устройства

## Запуск
```bash
docker compose up --build
```

После запуска: [http://localhost:8080](http://localhost:8080)
