# Требования к MVP
- Регистрация и авторизация пользователя
- Страницы с шаблонами резюме
- Сохранение информации в заполненных полях при переходах между шаблонами
- Сохранение заполненных шаблонов в личном кабинете
- Загрузка собственных файлов и их отображение рядом с шаблоном для переноса
- Экспорт сохранённых резюме в PDF
